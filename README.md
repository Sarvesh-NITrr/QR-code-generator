# QR-code-generator
Generates QR codes from user-provided text or URLs

Features

  Generate QR codes from text or URLs
  Multiple QR code size options (100×100 to 500×500)
  Instant QR code preview
  Download generated QR codes as PNG images
  Simple and responsive user interface
  Technologies Used
  HTML5 – Structure of the application
  CSS3 – Styling and layout
  JavaScript (Vanilla JS) – Logic and DOM manipulation
  QRCode.js – Library used for QR code generation

How It Works
  Enter any text or URL in the input field.
  Select the desired QR code size.
  Click the Generate button to create the QR code.
  Click Download to save the generated QR code as an image.
